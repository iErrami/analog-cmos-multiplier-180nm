# Analog CMOS Multiplier Design (180 nm)

## Overview

This project presents the design and simulation of an **Analog CMOS Multiplier** implemented using **180 nm CMOS technology** in **Cadence Virtuoso**.

The circuit was designed at the transistor level using **NMOS** and **PMOS** transistors. Its architecture is based on **differential pairs**, **current mirrors**, and an **output stage** to perform analog multiplication.

## Objectives

* Design an analog CMOS multiplier at the transistor level.
* Validate the circuit through simulation.
* Analyze the relationship between the input signals and the output voltage.

## Design Flow

1. Transistor-level schematic design.
2. Simulation testbench implementation.
3. DC and transient simulations.
4. Performance verification and analysis.
